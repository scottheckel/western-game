western-game
============

A little top-down western themed game created in two hours using Quintus (http://www.html5quintus.com).

Play the game at http://www.scottheckel.com/games/western

Controls
--------
(requires Gamepad and Chrome)
Left Stick - Move
Right Stick - Aim
RB - Shoot
